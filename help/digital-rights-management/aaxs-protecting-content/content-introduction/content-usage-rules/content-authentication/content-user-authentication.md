# Authentification de l’utilisateur {#user-authentication}

Indique si des informations d’identification, telles que le nom d’utilisateur et le mot de passe, sont requises pour acquérir une licence. Si une licence authentifiée (basée sur l’identité) est spécifiée, le serveur authentifie l’utilisateur avant d’émettre une licence.

**Exemple de cas pratique :** Un service d’abonnement peut nécessiter la saisie d’un nom d’utilisateur/d’un mot de passe avant de délivrer une licence de contenu. Un DVD ou un disque Blu-ray avec une copie numérique peut fournir un code ou un autre jeton comme preuve de paiement, qui peut être consommé pour un téléchargement électronique.
