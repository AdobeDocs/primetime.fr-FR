# Supprimer des plages{#delete-ranges}

Vous pouvez supprimer les périodes entre le début et la fin dans l’heure locale de la chronologie.

>[!TIP]
>
>Pour supprimer uniquement certaines plages du contenu, créez une `CustomRangeMetadata` et indiquez le type en tant que `DELETE` avec les plages personnalisées définies.

La carte publicitaire doit être utilisée comme définie par le serveur de publicités.

1. Pour supprimer des plages avec une publicité de prise de décision publicitaire Adobe Primetime :

   ```
   {   
       "properties": [],
       "stream": {
           "manifests": [
               {
                   "url": "https://d398890tia84ty.cloudfront.net/e2e-vod/cloudfront_vod_hls_tos_30fps.m3u8",
                   "type": "hls"
               }
           ],
           "metadata": {
               "time-ranges": {
                   "type": "delete",
                   "time-range-list": [
                       {
                           "begin": 0,
                           "end": 20000
                       },
                       {
                           "begin": 69000,
                           "end": 99000
                       },
                       {
                           "begin": 251000,
                           "end": 281000
                       },
                       {
                           "begin": 514000,
                           "end": 544000
                       }
                   ]
               },
               "ad": {

                       {

                       }
                   ],
   ```
